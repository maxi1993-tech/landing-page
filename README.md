# Landing Page

A landing page project built as part of [The Odin Project](https://www.theodinproject.com/) Foundations course.

## Description

This project was built from scratch by following a provided design mockup. The goal was to practice HTML structure and CSS styling independently, without any starter code.

## LIVE demo 

[View on GitHub Pages](https://maxi1993-tech.github.io/landing-page/)

## Built With 

- HTML5
- CSS3
- Flexbox
- Google Fonts (Roboto)

## What I learned 

- Structuring a page with semantic HTML
- Using Flexbox for layout
- Applying CSS custom properties (variables)
- Working with containers to control width while full-width backgrounds
- Commiting regularly with Git
